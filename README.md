Proyek ini adalah aplikasi CRUD Data Buku Perpustakaan berbasis PHP Native dan MySQL yang digunakan untuk mengelola informasi buku secara sederhana dan efisien. Aplikasi ini menyediakan fitur untuk menambah, menampilkan, mengedit, dan menghapus data buku yang terdiri dari judul, penulis, penerbit, dan tahun terbit. Seluruh data ditampilkan dalam bentuk tabel sehingga mudah dibaca dan dikelola oleh pengguna. Menu Tambah Buku memudahkan proses input data baru, sedangkan fitur edit dan hapus membantu menjaga data tetap akurat. Dengan tampilan yang sederhana dan fungsional, proyek ini cocok sebagai latihan dasar pemahaman konsep CRUD dalam pengembangan aplikasi web menggunakan PHP dan MySQL.
